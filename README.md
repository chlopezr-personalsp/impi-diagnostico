# Autodiagnóstico IMPI

Índice de Madurez del Portafolio de Infraestructura — herramienta de autoevaluación
basada en la metodología **MeDIPI: Infraestructura como Portafolio**.

En 12 preguntas, un administrador de infraestructura obtiene su nivel de madurez
(Reactivo → Básico → Gestionado → Estratégico) y un informe PDF con prioridades.

Publicado con GitHub Pages.
